## About 

Web-app as a promo for cookbook. Organize your diet, create food plans, manage your day to day food fight to be more healthy.
Basic requirements:
* adding, modifying recipes;
* show recipes list;
* create, show, edit, food schedule;
* user page
* admin page

Additonal requirements:
* voting for recipes


## Techs
- Python 3.6
- Django 3
- PostgreSQL
- HTML/CSS/Bootstrap
